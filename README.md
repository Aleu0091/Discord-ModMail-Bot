# Discord-ModMail-Bot

# How to Run

### Installation

To install this project, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/Aleu0091/Discord-ModMail-Bot.git
    ```

2. Install the required modules:

    ```bash
    npm install
    ```

### Running the Bot
1. Execute the script:

    ```bash
    node index.js
    ```


## Requirements

-   **Node.js**
